# 🎨 AI Image Prompts — Sanatan Shiv Shakti

A reusable prompt kit for generating **banners, posters, invitation cards, social posts,
ID cards, volunteer cards, and deity images** that match the website's look and feel.
Works with **any** AI image generator.

## 30-second how-to

1. Open the template you need (e.g. `poster.md`).
2. Copy the prompt, replace every `[BRACKETED]` value with your details.
3. Paste into your AI image tool.
4. **For Hindi/Bengali text or exact dates:** generate the artwork *without* text, then add
   the text in **Canva** (AI tools can't spell Devanagari/Bengali reliably).

## Recommended tools (2026)

- **Text on the image (English)** → **Ideogram v3** (best at in-image text), then GPT-Image.
- **Deity / scene artwork** → **Midjourney v7**, **FLUX 1.1 Pro**, or **Google Imagen**.
- **Easy & conversational** → **GPT-Image (ChatGPT)** or **Gemini / Imagen**.
- **Add Hindi/Bengali text, logo, exact layout** → **Canva** on top of the AI artwork.
