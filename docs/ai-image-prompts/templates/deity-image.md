# Template — Deity Image (for reuse in banners, posters, cards)

Generate a clean deity image with a simple background so you can reuse it across many
designs. Prefer **traditional illustration / calendar-art** style over photorealism for
deities. **1:1** or **3:4**.

```
Create a respectful, traditional DEVOTIONAL IMAGE of [DEITY] for a Hindu spiritual
organization. Calendar-art / temple-art illustration style, dignified and reverent.

[DEITY] in a calm, front-facing devotional pose with a soft golden halo and gentle divine
glow. Correct traditional attributes: [ATTRIBUTES]. Warm, auspicious lighting. Simple,
softly blurred saffron–gold background (or plain background for easy cut-out). Ornamental
but uncluttered. Highly detailed, print quality.

AVOID: distorted or extra limbs, malformed face/hands, casual or comic poses, mixing
attributes of different gods, irreverent or disrespectful depiction, garbled text,
watermark.
```

**Attribute quick-reference (fill `[ATTRIBUTES]`):**

| Deity | Attributes to request |
|-------|------------------------|
| **Hanuman Ji** | gada (mace), orange/sindoor body, devoted posture, sometimes Sanjeevani mountain or hands folded |
| **Lord Shiva** | trishul, crescent moon, Ganga in hair, blue throat, rudraksha, meditative on Kailash |
| **Lord Ram & Sita (Ram–Janki)** | Ram with bow, both serene and noble, royal yet gentle |
| **Lord Krishna** | flute, peacock feather, yellow dhoti, gentle smile |
| **Goddess Durga / Shakti** | multiple arms with weapons, lion/tiger, serene powerful expression |

> Tip: ask for the deity **without any text**, then place it into your poster/banner/card
> template in Canva.
