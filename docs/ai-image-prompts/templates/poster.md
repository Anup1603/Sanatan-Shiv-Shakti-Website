# Template — Event Poster (3:4 portrait)

Ready-to-paste. Fill the `[BRACKETS]`. For Hindi/Bengali text, generate the artwork
without text (Workflow A in the Master Prompt) and add text in Canva.

## Generic poster prompt

```
Create a premium 3:4 portrait event POSTER for "Sanatan Shiv Shakti", a non-political
spiritual & charitable youth organization in Howrah, West Bengal.

SUBJECT: [DEITY OR SCENE — e.g. Lord Hanuman in a reverent devotional pose with a soft
golden halo].
COMPOSITION: deity/scene in the centre; a clean decorative band at the TOP for the
organization name and a clean band at the BOTTOM for date/time/venue. Ornamental gold
mandala border. Generous negative space for text.
STYLE: premium, modern, devotional, elegant, clean. Colors: deep saffron #C2410C,
bright saffron #F59324, deep maroon #7A1E27, temple gold #C8A248, warm cream #FFFAF2.
Motifs: lotus, lit diyas, marigold garland, soft gold light rays. Auspicious golden
lighting, gentle glow, ultra detailed, print quality.

TEXT (English only; omit if your tool can't spell — add in Canva):
  Sanatan Shiv Shakti — Seva · Sanskar · Sanskriti
  [EVENT TITLE]
  [DATE] ([DAY]) · [TIME]
  [VENUE]
  [SHORT PROGRAM LINE]

AVOID: clutter, distorted faces/hands, garbled text, watermark, cartoonish or
disrespectful deity depiction, political symbols.
```

## ✅ Worked example — Sundar Kand Path (16 June 2026)

```
Create a premium 3:4 portrait event POSTER for "Sanatan Shiv Shakti", a non-political
spiritual & charitable youth organization in Howrah, West Bengal.

SUBJECT: Lord Hanuman in a calm, reverent devotional pose holding his gada (mace), warm
orange/sindoor tone, a soft divine golden halo, traditional calendar-art style, dignified
and respectful.
COMPOSITION: Hanuman centred; clean cream band at top for the org name, clean band at the
bottom for event details; ornate gold mandala border; lotus and lit diyas in the lower
corners. Plenty of clear space for text.
STYLE: premium, modern, devotional, elegant, clean. Colors: deep saffron #C2410C, bright
saffron #F59324, deep maroon #7A1E27, temple gold #C8A248, warm cream #FFFAF2. Soft
auspicious golden lighting, gentle glow, ultra detailed, print quality.

TEXT (English; or generate without and add in Canva):
  Sanatan Shiv Shakti — Seva · Sanskar · Sanskriti
  Hanuman Ji's Sundar Kand Path
  16 June 2026 (Tuesday) · 4:00 PM
  Ram Janki Mandir, Shibpur, Howrah
  Sundar Kand Path · Aarti · Prasad — All devotees welcome

AVOID: clutter, distorted faces/hands, garbled text, watermark, extra limbs, cartoonish
or disrespectful depiction, political symbols.
```
