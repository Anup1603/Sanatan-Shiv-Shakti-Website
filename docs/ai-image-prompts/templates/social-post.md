# Template — Social Media Post (1:1) & Story/Status (9:16)

## Square post (1:1) — Instagram / Facebook feed

```
Create a premium 1:1 square SOCIAL MEDIA POST for "Sanatan Shiv Shakti", a spiritual &
charitable youth organization in Howrah, West Bengal.

SUBJECT: [DEITY OR SCENE — e.g. a glowing diya with lotus petals / Hanuman Ji emblem].
COMPOSITION: bold central focus; a clean strip at top or bottom for a short headline;
gold mandala accents; subtle saffron-to-maroon gradient background.
STYLE: premium, modern, devotional, eye-catching, scroll-stopping but tasteful. Colors:
saffron #C2410C / #F59324, maroon #7A1E27, gold #C8A248, cream #FFFAF2. Soft golden glow,
high detail.

TEXT (very short, English; add in Canva for Hindi/Bengali):
  [SHORT HEADLINE — e.g. "Jai Hanuman 🙏"]
  Sanatan Shiv Shakti · [DATE / TIME if event]

AVOID: clutter, garbled text, watermark, distorted faces, political symbols.
```

## Story / WhatsApp status (9:16 portrait)

Use the same prompt but set **ASPECT RATIO: 9:16 (vertical)** and keep the focal subject
in the **centre third** (top and bottom get covered by app UI). Reserve the middle for one
short line of text.
